# Fashion MNIST Classification using Artificial Neural Networks (ANN)

This repository contains a Deep Learning project that classifies images of clothing and accessories from the **Fashion MNIST** dataset. The model is built using **TensorFlow** and **Keras** via an Artificial Neural Network (ANN) architecture.

## 🚀 Project Overview
The goal of this project is to build a multi-class classification model that can recognize 10 different categories of fashion items (e.g., T-shirts, Trousers, Sneakers, Dresses).

### Key Features:
* **Data Preprocessing:** Scaling pixel values for faster convergence.
* **Architecture:** Multi-layered Artificial Neural Network (Dense layers).
* **Visualization:** Training/Validation loss and accuracy plots, and visual prediction results.
* **Evaluation:** Assessing the model performance on unseen test data.

## 🛠 Tech Stack
* **Language:** Python
* **Libraries:** TensorFlow, Keras, NumPy, Matplotlib
* **Environment:** Google Colab

## 📊 Dataset
The dataset used is **Fashion MNIST**, which consists of:
* **60,000** training images.
* **10,000** test images.
* **Image size:** 28x28 pixels (Grayscale).
* **Classes:** T-shirt/top, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot.
